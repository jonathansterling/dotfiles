# dotfiles

- Run `./setup.sh` to symlink these dotfiles into your home directory.
- Each top-level directory (`bash/`, `nvim/`, `tmux/`) is a Stow "package".

## Dependencies

[Stow](https://www.gnu.org/software/stow/)

### Linux
```bash
sudo apt install stow
```

### macOS
```bash
brew install stow
```
